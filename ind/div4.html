Objects

Objects are collections of key-value pairs:

Example:


const person = {
    name: "Zoro",
    age: 25,
    profession: "Developer"
};

// Accessing properties
console.log(Object.keys(person)); // Output: ["name", "age", "profession"]
console.log(person.name); // Output: Zoro
